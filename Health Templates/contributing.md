# Contributing to {{project}}

Thanks for your interest in contributing! 🎉

## Getting started

1. Fork the repository and clone your fork.
2. Create a branch: `git checkout -b feature/my-change`.
3. Make your changes and add tests where relevant.
4. Run the test suite and make sure everything passes.
5. Commit with a clear message and open a pull request.

## Reporting bugs

Open an issue and include steps to reproduce, expected vs. actual behavior,
and your environment (OS, version, etc.).

## Code style

Please match the existing code style and keep changes focused. One logical
change per pull request makes review easier.

## Code of Conduct

By participating, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).
