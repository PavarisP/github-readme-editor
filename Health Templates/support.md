# Support

Need help with {{project}}? Here's how to get it:

- 📖 Read the [README](README.md) and documentation first.
- 🔍 Search existing [issues](../../issues) — your question may be answered.
- 💬 Open a new [discussion](../../discussions) for questions and ideas.
- 🐛 Open an [issue](../../issues/new) for bugs and feature requests.

Please avoid emailing maintainers directly for support questions so answers can
help the whole community.
